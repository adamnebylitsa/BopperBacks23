# Wii Love Minecraft
## HopperHacks MR PRESIDENTS

Minecraft mod that allows you to use a Wii remote with nunchuck to play Minecraft.

## Wii Remote Control
Destroy Block -Move the Wii Remote Down   
Attack -Tilt the Wii Remote to the Right
